# Prompt English Coach Implementation Folder

This folder captures the product goal, technical design, and execution plan for building Prompt English Coach as a full working Claude Code plugin.

Read in order:

1. `01-goal.md` - product goal, positioning, scope, and definition of done.
2. `02-design.md` - plugin architecture, hook behavior, mode semantics, and risks.
3. `03-execution-plan.md` - task-by-task implementation plan for a future execution pass.
4. `04-execution-report.md` - completed work and verification evidence from implementation.

The implementation target is a distributable Claude Code plugin marketplace repository, not a thin MVP prototype.
